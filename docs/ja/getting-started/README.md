## 概要

wasm-num は Lean 4 ライブラリです。Lean プロジェクトに Lake 依存関係として追加して使用します。必要なもの：

- **Lean 4**（v4.29.0-rc6 またはそれと互換性のあるバージョン）
- **Lake**（Lean 4 にバンドル）
- 初回の Mathlib キャッシュダウンロードのためのネットワークアクセス
