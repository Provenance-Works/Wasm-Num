## Overview

wasm-num is a Lean 4 library — you use it by adding it as a Lake dependency to your Lean project. It requires:

- **Lean 4** (v4.29.0-rc6 or compatible)
- **Lake** (bundled with Lean 4)
- Network access for initial Mathlib cache download
